* Org mode shortcuts
- tangle :: ~org-babel-tangle~, ~C-c C-v C-t~

** Config
- ~C-c C-c~ :: Refresh local setup for that file

** Dates:
- ~C-c .~ :: active timestamps
  creates an entry in the agenda
- ~C-c !~ :: inactive timestamps
  doesnÂ´t create an entry in the agenda
- link :: [[https://orgmode.org/orgguide.html#Timestamps][timestamps]]

** Links:
- ~C-c C-l~ :: create link
- link :: [[https://orgmode.org/orgguide.html#Hyperlinks][hyperlinks]]

** Tables
- ~C-c }~ :: display cells references
- ~C-c ?~ :: display ref for that cell
  references are =@ROW$COLUMN=
- link :: [[https://orgmode.org/manual/References.html][table references]]
- formula :: =@I..II=
  select all the rows between horizontal lines (hline) 1 and 2
  =vmean(@I..II);EN= column range mean, treats empty values as null (0)

** Whitespace
- ~M-x whitespace-mode~ :: Activates a better whitespace mode
- ~M-x auto-fill-mode~ :: Auto-fill, enforce max size limit
- ~M-q~ :: Force auto-fill on current paragraph

** Feetnotes
- ~C-c C-x f~ :: Creates a footnote, see [[https://orgmode.org/guide/Creating-Footnotes.html][guide]].
- ~C-c C-c~ :: to jump between definition and refs.

** Todos
- ~C-c C-t~ :: cycling throught the states of the todos
- ~C-u C-c C-t~ :: Log the change of status

** Clocking work
- ~C-c C-x C-i~ :: Clock in
- ~C-c C-x C-o~ :: Clock out
- ~C-c C-x C-d~ :: Display clocks

** Includes files
Start with the directive ~INCLUDE~ and the location of the file

  - ~C-c &~ :: Go back to the original doc
